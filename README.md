# Earn2Keep Landing Page

This is the "Coming Soon" landing page for **earn2keep.com**.

**Tagline:** *Earn it. Keep it.*

## Files in this project

- `index.html` — the main page
- `styles.css` — the design / styling
- `script.js` — handles the email signup form
- `favicon.svg` — the small icon shown in browser tabs

## How it works

When someone visits earn2keep.com, they see:
1. The Earn2Keep brand and "Earn It. Keep It." headline
2. A description of the platform
3. An email signup form to get notified at launch
4. A "How it works" 3-step explanation
5. A "Built for" list showing target audiences

For now, the signup form saves emails to the visitor's browser only (this is temporary). In a future phase, we'll connect this to Supabase so signups are saved to a real database that you can see and export.

## Notes for future updates

- Brand colors are placeholders (navy + green). They live in `styles.css` at the top, in the `:root` section, and can be easily changed later.
- All sections are mobile-responsive.
- No tracking, no cookies, no third-party scripts at this stage.
